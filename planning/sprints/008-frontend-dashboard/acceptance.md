# Sprint 008 Acceptance Criteria

This sprint is complete when:

- [ ] **Clinical Assistant UI:**
  - Users can enter a prompt and receive a response (AI or "Needs Review" status).
  - Citations are correctly displayed when provided by the backend.
- [ ] **HITL Dashboard:**
  - A list of pending reviews is displayed.
  - Clicking a review opens a detail view with all necessary context (Prompt, Response, Reason).
  - Resolution actions (`Approve`, `Edit`, `Reject`) can be triggered.
- [ ] **Integration:**
  - Frontend successfully communicates with the AI Gateway service.
- [ ] **State:**
  - `planning/STATE.md` is updated.
