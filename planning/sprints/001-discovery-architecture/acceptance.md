# Sprint 001 Acceptance Criteria

This sprint is complete when:

- [ ] **Foundational Data Layer:**
  - Architecture diagram shows data flow from Raw to Gold layers.
  - CMEK and VPC-SC perimeter strategies are documented.
  - Core clinical data schemas (FHIR-aligned) are drafted in `DATA_MODEL.md`.
- [ ] **AI Gateway Evaluation Gates:**
  - Proxy/Middleware architecture is defined.
  - PHI detection strategy (using GCP Sensitive Data Protection) is documented.
  - Evaluation metrics (Grounding, Hallucination, Safety) are defined with thresholds.
  - HITL trigger conditions are clearly specified.
- [ ] **Governance:**
  - Audit logging schema for AI interactions is defined.
- [ ] **Project State:**
  - `planning/STATE.md` reflects the completion of these priority specifications.
